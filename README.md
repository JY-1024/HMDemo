# HMDemo
这是一个练习鸿蒙开发的demo
----
1. Tabs 自定义底部导航栏
2. Swiper 轮播图的使用
3. Grid网格布局
4. Column、Row线性布局
5. @Builder全局自定义构建函数，只能自己使用
6. @Extend(Text)全局自定义样式
7. 页面生命周期aboutToDisappear
8. prompt.showToast 提示的使用
9. setTimeout() 定时器的使用
10. 路由页面跳转及参数传递
11. 国际化的使用
